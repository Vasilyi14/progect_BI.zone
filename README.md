Описание проекта
Название проекта: SiteAnalyzer
Краткое название: Анализатор сайтов с проверкой безопасности

Что делает программа:
SiteAnalyzer — это простое веб-приложение, которое позволяет быстро получить информацию о любом сайте и провести базовый анализ его безопасности.
Основные возможности:
•  Приём URL от пользователя через удобный веб-интерфейс
•  Автоматический анализ сайта:
	•  Получение заголовка страницы (<title>)
	•  Получение описания страницы (meta description)
	•  Определение сервера
	•  Проверка статуса ответа
•  Анализ безопасности (базовый):
	•  Проверка использования HTTPS / HTTP
	•  Проверка важных Security Headers (X-Frame-Options, CSP, HSTS и др.)
	•  Обнаружение распространённых уязвимостей (Clickjacking, слабая защита от XSS)
•  Вывод результатов в удобной красивой странице
•  Сохранение результатов анализа в JSON-файл (для дальнейшей обработки или записи в базу данных)

Техническая архитектура:
•  Backend: Go (язык программирования)
•  Frontend: Простой HTML + CSS + JavaScript (одна страница)
•  Коммуникация: HTTP (REST)
•  Хранение данных: JSON-файл (last_analysis.json) + готовность к интеграции с SQLite
•  Порт: 8080

Как работает:
1.  Пользователь открывает index.html
2.  Вводит ссылку на сайт
3.  Нажимает кнопку «Получить информацию»
4.  Go-сервер анализирует сайт
5.  Пользователя автоматически переводит на страницу с полным отчётом

Планируемые улучшения (можно добавить позже):
•  Запись всех анализов в SQLite базу данных
•  История проверенных сайтов
•  Более глубокий анализ (SSL, возраст домена, технологии сайта)
•  Экспорт отчёта в PDF
•  Проверка на наличие известных уязвимостей (CVE)
